# :computer: How to run

```bash
# Clone Repository
$ git clone https://github.com/Gustavohsdp/ignite-template-componentizando-a-aplicacao
```

```bash
# Install Dependencies
$ yarn

# Run Application
$ yarn start
$ yarn server
```
Go to http://localhost:8080/ to see the result.
